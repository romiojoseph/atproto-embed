AT Proto Embed

Add Bluesky post embeds or a comment section to your blog or website with a single script tag. Works with any AT Protocol client that follows the standard URL pattern.

Public posts are accessible to everyone regardless of which client you use or your login status.
  • No login or installation needed
  • No trackers or analytics
  • No content visibility issues
It's open source and entirely vibe coded.

This project exists because I needed it. I was trying to embed a Bluesky post on my blog, but the official script wouldn't load it because I had "Logged-out visibility" enabled (Which means Bluesky will honor this request, so it won't load it for logged out users). Fair feature, wrong tradeoff for me.

I looked at other solutions, but wanted something of my own. So I vibe coded it.

I had built a comment widget a while back, but revisiting it now, it felt unfinished. I had learned enough to do it somewhat good this time. And I wanted to leave that project untouched because I noticed that some people are using it.

The current project - AT Proto Embed - is not just a comment widget. It can embed any post from any client that follows the same Bluesky post link pattern (at://did:plc:abc1def2ghi3jkl/app.bsky.feed.post/rkey). The goal was simple, give full control over what shows and what doesn't. Browsers without native HLS support (like Firefox) automatically load hls.js on demand. No extra setup required.

Entirely vibe coded. Use it via CDN or grab the JS and CSS.
